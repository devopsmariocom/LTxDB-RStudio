# LTxDB-RStudio

A customized RStudio Docker image based on `docker.io/rocker/rstudio:4.5.1` with PostgreSQL support and common data science R packages pre-installed.

## Features

- PostgreSQL database connectivity (libpq-dev, RPostgres)
- Pre-installed R packages: tidyverse, ggplot2, rmarkdown, workflowr, BiocManager, and more
- Ready-to-use RStudio environment for data analysis and database work
